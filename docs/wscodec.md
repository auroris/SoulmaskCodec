## wscodec
wscodec: pure-JS codec for Soulmask actor_data property streams.Public surface re-exports - `import { UnrealBlob, FName, FGuid, ... }from 'wscodec';` works without reaching into individual submodules.Wire layout, top-level API, and Soulmask actor_data envelope aredocumented in `blob.mjs`. Property class hierarchy and the recursivefromReader/toBytes pattern are documented in `property.mjs`.
